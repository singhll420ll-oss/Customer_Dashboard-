# BiteMeBuddy - Food Delivery Web Application

A full-stack food delivery web application built with Python Flask and PostgreSQL.

## Features

- User authentication (mobile-based registration/login)
- Service catalog with discounts
- Menu items management
- Shopping cart with quantity control
- Order management with tracking
- User profile with location
- Real-time notifications
- Responsive mobile-first design

## Tech Stack

- **Backend**: Python 3.12+, Flask
- **Database**: PostgreSQL
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Templates**: Jinja2
- **Authentication**: bcrypt password hashing
- **APIs**: Geolocation API
